# User List App

A small web app built with plain JavaScript that fetches users from an API, displays them in a list, allows searching, and shows a loading indicator.

## Features

- Fetch users from [JSONPlaceholder](https://jsonplaceholder.typicode.com/users) using async/await.
- Display users dynamically in a list with name and email.
- Real-time search filter using `.filter()`.
- Show a "Loading..." message while fetching data.
